# Diabetes Diagnosis

### Live Application : [View](https://diadet.herokuapp.com/)

A simple web application that predicts if a person is diabetic or not with the help of given input values:
* Age
* Fasting Plasma Glucose
* Diastolic BP
* Systolic BP
* BMI 
* Average Blood Sugar
* Waist Circumference
* Hip Circumference
* Cholestrol
* OGTT
